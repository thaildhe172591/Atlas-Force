# /clean-code

Pragmatic coding standards. When this command is used, I will apply clean code principles to your request.

## Core Rules

1. **SRP**: Single Responsibility Principle
2. **KISS**: Keep It Simple, Stupid
3. **DRY**: Don't Repeat Yourself
4. **YAGNI**: You Aren't Gonna Need It

## Usage

Use this command when you want me to refactor code or implement a feature with extreme focus on code quality and simplicity.

```bash
/clean-code refactor this component
/clean-code implement a simple user service
```
