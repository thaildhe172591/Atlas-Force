# /code-review

Expert code review for quality, security, and performance.

## Focus Areas

1. Correctness & Edge Cases
2. Security (OWASP, Secrets, Injection)
3. Performance (N+1, Re-renders, Caching)
4. Code Quality (Naming, DRY, SOLID)
5. AI-Specific (Hallucinations, Prompt Safety)

## Usage

```bash
/code-review review this file
/code-review check my latest changes for security
```
