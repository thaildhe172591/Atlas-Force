# /systematic-debugging

Expert troubleshooting and root cause analysis.

## 4-Phase Logic

1. **Discovery**: Gather evidence, logs, and trace.
2. **Analysis**: Narrow down root causes.
3. **Execution**: Implement the fix.
4. **Verification**: Prove the fix works with evidence.

## Usage

```bash
/systematic-debugging fix the 500 error on payment
/systematic-debugging investigate high memory usage
```
